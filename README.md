# React ui-style Storybook 프로젝트

### Dev stack
- typescript
- rollup
- react
- babel

### Project start
- yarn
```
// build
yarn build
// development
yarn start
```
- npm
```
// build
npm build
// development
npm start
```

### remove
```
yarn clear
```

### reinstall
```
yarn reinstall
```

### peerDep
```
yarn add -D --ignore-scripts install-peers react react-dom
```
